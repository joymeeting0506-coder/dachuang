# 实验记录与结果分析

## 一、实验目标

本实验记录用于整理非遗纹样生成模型的训练过程、参数设置、生成效果和对比分析。重点关注 SDXL + LoRA 在小样本非遗纹样生成中的表现，并使用 StyleGAN2 作为对比实验。

核心问题：

- 模型是否学到了指定非遗纹样风格？
- 小样本数据下是否容易过拟合？
- 生成结果是否清晰、多样、适合文创设计？
- SDXL + LoRA 与 StyleGAN2 的效果差异是什么？

采用半自动标注的原因：

- 非遗纹样数据量较大，全部人工精细标注工作量过高。
- 本项目主要目标是跑通模型微调和平台原型，不需要达到专业数据集的细粒度标注强度。
- LoRA 微调更依赖高质量图片和稳定描述文本，因此优先精标每类 50-100 张高质量样本。
- 低质量或版权不明确的图片不进入训练集，只作为调研参考。
