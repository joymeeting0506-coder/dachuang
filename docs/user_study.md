# 校内测试记录

## 一、测试目的

本测试用于验证“非遗纹样智能生成与商用设计平台”原型在校内场景中的可用性，重点观察用户是否能顺利完成纹样生成、参数调节和图片导出，并收集用户对生成效果和平台功能的反馈。

测试不追求大规模商业验证，主要服务于大学生创新训练项目的阶段性成果展示和最终报告。

## 三、测试准备

测试前需要准备：

- 可运行的 Web 平台。
- 至少 1-2 个可选非遗风格。
- 可生成或模拟返回的图片结果。
- 参数调节功能，例如类别、主题、颜色、生成数量。
- 图片导出功能，例如 PNG 或 JPG。
- 问卷或访谈记录表。

如果模型推理速度较慢，可以提前准备部分生成结果，测试时让平台调用样例图，重点验证交互流程。

## 四、测试任务

每位用户完成以下任务：

1. 打开平台首页。
2. 选择一种非遗纹样类别，例如剪纸或敦煌。
3. 输入或选择一个主题，例如花卉、云纹、莲花。
4. 调节至少一个参数，例如颜色、风格强度或生成数量。
5. 点击生成按钮。
6. 从生成结果中选择一张最满意的图片。
7. 导出图片。
8. 完成问卷或简短访谈。

## 五、问卷设计

建议使用 1-5 分量表：

| 分数 | 含义 |
| --- | --- |
| 1 | 非常不同意 / 非常差 |
| 2 | 不太同意 / 较差 |
| 3 | 一般 |
| 4 | 比较同意 / 较好 |
| 5 | 非常同意 / 非常好 |

问卷问题：

| 编号 | 问题 | 评分 |
| --- | --- | --- |
| Q1 | 我能理解平台的主要功能 | 1-5 |
| Q2 | 我能顺利完成纹样生成 | 1-5 |
| Q3 | 参数调节对生成结果有帮助 | 1-5 |
| Q4 | 生成结果具有明显非遗纹样风格 | 1-5 |
| Q5 | 生成结果比较美观 | 1-5 |
| Q6 | 生成结果适合用于文创产品设计 | 1-5 |
| Q7 | 图片导出功能容易使用 | 1-5 |
| Q8 | 我愿意继续尝试使用该平台 | 1-5 |

开放问题：

- 你最喜欢哪个生成结果？为什么？
- 你觉得哪个功能最有用？
- 你觉得平台哪里最难理解？
- 你希望增加哪些风格或功能？
- 如果用于校园文创，你会希望生成什么产品图案？

## 十、可放入项目报告的描述

本项目在校内邀请若干名同学对平台原型进行了试用测试。测试任务包括选择非遗纹样类别、输入生成需求、调节参数、生成图片并导出。通过问卷和访谈收集用户反馈，用于评估平台在校内文创设计场景下的可用性。测试结果将从任务完成率、生成效果评分、文创可用性和用户建议等方面进行分析，并作为后续系统优化依据。
