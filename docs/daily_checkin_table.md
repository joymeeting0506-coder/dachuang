# 14 天双人打卡表

## 使用说明

每天结束前，两名成员都需要填写当天打卡。填写时尽量写清楚具体产出，例如文件名、接口名、页面功能、数据数量、截图位置或测试人数。

建议每天填写时间不超过 10 分钟，但必须真实记录。
